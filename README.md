#Whatsapp chatbot backend(Spring boot)
This is a simple backend project that simulates a whatsapp chatbot using java and springboot.
## Features
-Rest API endpoint: '/webhook'
-Accepts POST request
-responds to message:
-"hi" -> "hello" how can I help you
-"bye" ->"goodbye"

## Tech Stack
-JAVA
-SPRING BOOT
-MAVEN

## How to Run
1.clone the repository
2.open in Intellij
3.Run 'chatbotApplication.java'
4.use postman to test the API
